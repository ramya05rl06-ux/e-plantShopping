# Paradise Nursery Shopping Cart Application

## 🌟 Project Description
Paradise Nursery is a responsive, Single-Page Application (SPA) designed as an online houseplant store. Built utilizing React.js for the interface and Redux Toolkit for state management, it provides a seamless plant browsing and shopping experience. 

## 🛠️ Main Component & File Details

1. README.md: Contains the core project name, overview, and architectural layout details.
2. App.jsx: The main entry controller that manages screen switching between the welcome Landing Page and the main Product Directory.
3. App.css: Holds layout structures and responsive styling, including the background image configuration for the Paradise Nursery storefront.
4. AboutUs.jsx: Renders descriptive details about the company's background, environmental values, and mission statement.
5. CartSlice.jsx: Handles global state management with three core reducer functions: `addItem()`, `removeItem()`, and `updateQuantity()`.
6. ProductList.jsx: Displays the full catalog of 18 houseplants organized by unique categories, complete with product photos and actionable "Add to Cart" options.
7. CartItem.jsx: Renders the active shopping cart layout with immediate pricing totals and item management controls.

## 💻 Tech Stack Used
* Framework: React.js (Functional Components & Hooks)
* State Management: Redux Toolkit (Slices & Actions)
* Bundler: Vite
* Styling: Pure CSS3
